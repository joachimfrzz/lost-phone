## Écrans à capturer (suggestion test)

Minimum pour un brief utile (~8–10 captures) :

1. Liste des discussions (accueil)
2. Conversation 1:1
3. Conversation de groupe
4. Profil / détails contact
5. Appels / historique
6. Recherche
7. Paramètres / « Vous » (si pertinent jeu)
8. Panneau pièces jointes ou réactions (optionnel)
