# Lost Phone — Apps utiles (vision scénariste)

> Liste des applications **utiles à toutes les histoires Lost Phone possibles**.  
> Une app est développée si elle sert **au moins un archétype narratif** récurrent.  
> Chaque histoire LPSP choisit un **sous-ensemble** via `manifest.apps_presentes`.

**Règles produit**
- Écrans IN = quasi identiques iOS
- Contenu = 100 % LPSP
- Banque = **1 template**, branding/couleurs par LPSP (`banque.nom`, `banque.theme`)
- SNCF Connect = **confirmé** (la-nuit-du-train + futurs récits voyage)

  "contenu": {
    "marque": "Crédit Agricole",
    "theme": {
      "primary": "#009597",
      "logo": "credit-agricole"
    },
    "comptes": [{ "nom": "Compte courant", "solde": "1 234,56 €" }],
    "operations": [{ "libelle": "...", "montant": -42.0, "date": "..." }]
